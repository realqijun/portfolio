cd

Usage:
cd <arg>
cd <no arg> == cd $HOME (note: if $HOME is unset exit status 1)
cd <arg1> <arg1> -> exit status 2
cd <dir name over 255> -> exit status 1
cd <non existent dir> -> exit status 1

echo

Usage:
echo <arg1> <arg2> ...
prints strings
no known exit statuses

pwd

prints working directory using getcwd function
minishell ver: ignore args
no known exit statuses

env

prints current environment
minishell ver: ignore args
no known exit statuses

unset

unset <arg1> <arg2> ...
unsets environment variables
no known exit statuses

export

Usage:
export <arg1> <arg2> ... (arg: <string>=<string>/ <string>+=<string>)
exports environment variables
no known exit statuses
